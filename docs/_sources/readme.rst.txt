========
Overview
========

UFEDMM is an OpenMM extension for applying the Unified Free Energy Dynamics method.

* Free software: MIT license

Installation
============

::

    git clone https://github.com/craabreu/ufedmm.git
    cd ufedmm
    python setup.py install

=============
